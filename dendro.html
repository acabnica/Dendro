Dendro Element

Dendro is an element of nature. It currently only has one reaction. The reaction in question is burning.

When Pyro collides with Dendro, Burning starts. This effect can go through a lot of HP really quickly if not stopped.

Moreover,burning is unique because only Pyro characters can utilize this reaction. If you inflict any other element onto a Dendro status, it will simply stack or not stay on for long.

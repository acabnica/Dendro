Dendro Element

Dendro is an element of nature. Its associated Archon is Kusanali, whose domain is Sumeru.

What is Dendro?

Some enemies may create a Dendro shield or constructs, giving them great attacking or defensive skills. Breakable shields and constructions must be broken with Pyro applications. When Pyro collides with Dendro, Burning starts. This effect can go through a lot of HP really quickly if not stopped.

Moreover,burning is unique because only Pyro characters can utilize this reaction. If you inflict any other element onto a Dendro status, it will simply stack or not stay on for long.
